CSS: Styling Web Pages

CSS (Cascading Style Sheets) is used to style the presentation of HTML elements, controlling colors, fonts, and layout.

CSS Concepts

  • Selectors: Target HTML elements (e.g., `p`, `.my-class`, `#my-id`).
  • Properties: `color`, `font-size`, `margin`, `padding`, `background-color`.
  • Values: `color: blue;`, `font-size: 16px;`.
  • The Box Model: Content, padding, border, and margin.
  • Layout: `float`, `flexbox`, and `grid`.
/* Button Style */
.my-button {
    background: linear-gradient(to right, #64b5f6, #2196f3);
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    font-size: 1em;
}

.my-button:hover {
    background: linear-gradient(to right, #2196f3, #64b5f6);
}

JavaScript: Making Webpages Interactive

JavaScript adds interactivity and dynamic behavior to web pages.

JavaScript Concepts

  • Variables: `let`, `const`, and `var`.
  • Data Types: Numbers, strings, booleans, arrays, objects, `null`, `undefined`.
  • Operators: Arithmetic, comparison, logical.
  • Functions.
  • Events: `click`, `mouseover`, `load`.
  • DOM Manipulation.
  • Control Flow: `if/else`, `for`, `while`.

Flexbox

Flexbox is a powerful layout module.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 200px;
}

.item {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  margin: 10px;
}

CSS Grid

CSS Grid is a two-dimensional layout system.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  background-color: lightcoral;
  padding: 20px;
  text-align: center;
}

Transitions and Animations

Create smooth transitions and animations.

.box {
  width: 100px;
  height: 100px;
  background-color: lightgreen;
  transition: all 0.5s ease;
}

.box:hover {
  width: 150px;
  height: 150px;
  background-color: lightcoral;
}

Transforms

Apply 2D or 3D transformations.

.transform-box {
  width: 100px;
  height: 100px;
  background-color: lightyellow;
  transform: rotate(45deg);
}

Pseudo-classes and Pseudo-elements

Style based on state or insert content.

a:hover {
  color: red;
}

p::first-line {
  font-weight: bold;
}

Responsive Design with Media Queries

Adapt layout based on screen size.

@media (max-width: 768px) {
  body {
    font-size: 16px;
  }
}

CSS Variables (Custom Properties)

Define reusable values.

:root {
  --primary-color: coral;
}

h1 {
  color: var(--primary-color);
}
