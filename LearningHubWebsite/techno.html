CSS: Styling Web Pages

CSS (Cascading Style Sheets) is used to style the presentation of HTML elements, controlling colors, fonts, and layout.

CSS Concepts

  • Selectors: Target HTML elements (e.g., `p`, `.my-class`, `#my-id`).
  • Properties: `color`, `font-size`, `margin`, `padding`, `background-color`.
  • Values: `color: blue;`, `font-size: 16px;`.
  • The Box Model: Content, padding, border, and margin.
  • Layout: `float`, `flexbox`, and `grid`.
/* Button Style */
.my-button {
    background: linear-gradient(to right, #64b5f6, #2196f3);
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    font-size: 1em;
}

.my-button:hover {
    background: linear-gradient(to right, #2196f3, #64b5f6);
}

Flexbox

Flexbox is a powerful layout module.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 200px;
}

.item {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  margin: 10px;
}
.container { display: flex; justify-content: center; align-items: center; height: 200px; } .item { width: 100px; height: 100px; background-color: lightblue; margin: 10px; }

CSS Grid

CSS Grid is a two-dimensional layout system.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  background-color: lightcoral;
  padding: 20px;
  text-align: center;
}
.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; } .grid-item { background-color: lightcoral; padding: 20px; text-align: center; }

Transitions and Animations

Create smooth transitions and animations.

.box {
  width: 100px;
  height: 100px;
  background-color: lightgreen;
  transition: all 0.5s ease;
}

.box:hover {
  width: 150px;
  height: 150px;
  background-color: lightcoral;
}
.box { width: 100px; height: 100px; background-color: lightgreen; transition: all 0.5s ease; } .box:hover { width: 150px; height: 150px; background-color: lightcoral; }

Transforms

Apply 2D or 3D transformations.

  width: 100px;
  height: 100px;
  background-color: lightyellow;
  transform: rotate(45deg);
}
.transform-box { width: 100px; height: 100px; background-color: lightyellow; transform: rotate(45deg); }

}

Responsive Design with Media Queries

Adapt layout based on screen size.

@media (max-width: 768px) {
  body {
    font-size: 16px;
  }
}
@media (max-width: 768px) { body { font-size: 16px; } }

CSS Variables (Custom Properties)

Define reusable values.

:root {
  --primary-color: coral;
}

h1 {
  color: var(--primary-color);
}
:root { --primary-color: coral; } h1 { color: var(--primary-color); }

The Interactive Solar System

Explore our solar system with this interactive animation.

This section visually represents the basic orbital mechanics of a simplified solar system using CSS animations. The Sun is the central element, and two planets orbit around it at different speeds and distances. A moon also orbits one of the planets, demonstrating a hierarchical orbital system.

.solar-system-container { width: 400px; height: 400px; position: relative; margin: 20px auto; } .sun { width: 80px; height: 80px; background-color: #ffc107; border-radius: 50%; position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); box-shadow: 0 0 50px #ffc107; z-index: 1; } .planet { position: absolute; border-radius: 50%; border: 1px dashed #ccc; animation: orbit 20s linear infinite; transform-origin: 200px 200px; } .planet-inner { width: 30px; height: 30px; background-color: #1e88e5; border-radius: 50%; position: absolute; top: -15px; left: -15px; z-index: 2; } .planet2 { animation-delay: 5s; animation-duration: 25s; transform-origin: 200px 200px; } .planet2-inner { width: 20px; height: 20px; background-color: #f44336; border-radius: 50%; position: absolute; top: -10px; left: -10px; z-index: 2; } .moon { position: absolute; width: 10px; height: 10px; background-color: #aaa; border-radius: 50%; animation: moonOrbit 5s linear infinite; transform-origin: 15px 15px; left: 15px; top: -5px; z-index: 3; } @keyframes orbit { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } @keyframes moonOrbit { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } }
